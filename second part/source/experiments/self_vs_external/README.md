# Self vs External Annotations Comparison

An experiment designed to compare rules discovered based on the subject's self-reported emotions against those derived from objective or subjective labels provided by external observers. The goal is to highlight the convergences and differences in physiological rules between what individuals report feeling and how others perceive their physiological signals within the same dataset.

### Datasets & Annotations
- **Datasets:** K-emoCon (both), CASE, CEAP, EmoWorker (Self), EMBOA (External)
- **Annotations:** Both Self-annotations (participant self-report) and External-annotations (external observers' ratings) compared globally and directly block-by-block.
