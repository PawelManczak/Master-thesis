# Cross-Validation

This folder contains scripts aimed at machine-learning style rule generalization. It divides datasets into N-1 training sets and 1 validation set (leave-one-dataset-out process). It extracts common patterns learned from the training split and verifies what percentage of these rules "survive" and extrapolate successfully to the unseen validation set (hit rate and confidence variations).


### Datasets & Annotations
- **Datasets:** CASE, K-emoCon, CEAP, EmoWorker
- **Annotations:** Self-annotations (subjective participant ratings)
